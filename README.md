# CRM AI Assistant

**CRM AI Assistant** adalah AI chatbot untuk membantu analisis bisnis dan CRM secara otomatis. AI ini dapat menarik data dari berbagai sumber—API eksternal, database MySQL/Postgres/Mongo—dan memberikan insight cepat mengenai produk, leads, sales, tim, web traffic, finance, dan market trend.

---

## Fitur Utama

* **Produk:** Katalog, harga, stok, rating.
* **Leads & Customers:** Info kontak dan status follow-up.
* **Sales:** Total penjualan, diskon, item terjual.
* **Support / Tickets:** Status tiket pelanggan.
* **Campaign / Ads:** Impressions, clicks, CTR, budget.
* **Web Traffic:** Trafik dari berbagai platform.
* **Team Performance:** Deals closed & revenue per anggota.
* **Finance & Market Trend:** Revenue, expense, net profit, tren kategori produk.
* **AI Insight:** Prediksi penjualan, leads, risiko churn, dan saran strategi berbasis data nyata.
* **AI / NLP Provider:** Mendukung OpenAI, Gemini, dan Claude. NLP internal mendeteksi pertanyaan CRM secara fleksibel.

---

## Cara Pakai

1. Install dependencies:

```bash
npm install
```

2. Jalankan chat:

```bash
node app.js
```

3. Ketik pertanyaan CRM di terminal, ketik `exit` untuk keluar.

---
